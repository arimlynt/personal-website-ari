Project IOT

Ini adalah salah satu project IOT yang menggunakan mikrokontroller ESP8266 yang di gabungkan dengan sebuah web server yang dibuat dengan PHP. Jadi cara kerja project ini yaitu menampilkan nilai sensor LDR atau sensor cahaya ke web server. Sedangkan untuk cara kerja LDR sendiri yaitu, apabila sensor di beri cahaya maka lampu LED akan mati, jika tidak ada cahaya atau gelap maka lampu LED hidup.

Rest API E-Course

Ini adalah project rest api untuk website E-Course. Rest API ini akan di manfaat di sisi front end untuk di ambil data yang ada di dalamnya. Metode yang digunakan yaitu :
GET: Untuk menampilkan data,
POST: Untuk menambah data,
PUT: Untuk Mengubah data,
DELETE: Untuk Menghapus data.
Teknologi yang digunakan yaitu Node JS dengan framewok Express JS dan database Mongo DB.
